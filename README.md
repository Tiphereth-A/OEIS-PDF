# OEIS-PDF

![GitHub](https://img.shields.io/github/license/Tiphereth-A/OEIS-PDF)

[简体中文](doc/README-zh-hans.md)

Scrpits for generating PDF document of some sequences from [The On-Line Encyclopedia of Integer Sequences® (OEIS®)](https://oeis.org/).

Demo: <https://github.com/Tiphereth-A/OEIS-PDF/releases>

## Basic usage

1. List the content to be included in the documents in JSON format in the `data\detail` folder. The file name is ID of the sequence, and the content format is the same as JSON provided on the OEIS website.

    For example: If you want to include content with ID [A000004](https://oeis.org/A000004) and [A000108](https://oeis.org/A000108), then you should move the corresponding contents from <https://oeis.org/search?q=id:A000004&fmt=json> and <https://oeis.org/search?q=id:A000108&fmt=json> to `data\detail\A000004.json` and `data\detail\A000108.json` respectively, i.e.
